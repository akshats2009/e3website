Make Your First Open Source Pull Request

A beginner-friendly workflow, similar to firstcontributions/first-contributions.

  1. Fork this repository on GitHub

    Open akshats2009/e3website and click Fork so you get your own copy.

  2. Clone your fork locally 
    git clone https://github.com/<your-username>/e3website.git
cd e3website
  3. Create a branch for your change 
    git checkout -b add-my-first-contribution
  4. Make one simple change

    Examples: add your name in a contributor section, or improve one sentence in a page like about.html or outreach.html.

  5. Commit your update 
    git add .
git commit -m "docs: add my first contribution"
  6. Push your branch to your fork 
    git push origin add-my-first-contribution
  7. Open your pull request

    On GitHub, open your fork and click Compare & pull request. In your PR description, explain what you changed and why.

  8. Respond to feedback and merge

    If maintainers request updates, push commits to the same branch. Your pull request will update automatically.
